Este proyecto utiliza un entorno virtual de Python 
Recomendado para mantener separados los proyectos y sus librerías evitando incompatibilidades.
Utilizando la librería PyQt6 se modernizó el proyecto anterior LexerParser, consiguiendo muchas mejoras
tanto a nivel estético como a nivel usuario.

La bibliografía mas general usada para este proyecto se encuentra en los comentarios del src, sin tener
en cuenta foros de StackOverflow.
